1- Clone the repository
 git clone https://github.com/Mohamed-Waged/todo-list-app.git

2-Navigate to the project directory:
 cd todo-list-app

3-Install dependencies:
 composer install
 npm install

4-Set up the environment file:
 Copy code
 cp .env.example .env

5-Run migrations:
php artisan migrate

6-Serve the application:
php artisan serve

7-Compile assets:
npm run dev
